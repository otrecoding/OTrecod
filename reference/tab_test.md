# A simulated dataset to test the library

A dataset of 10000 rows containing 3 covariables and 2 outcomes.

## Usage

``` r
tab_test
```

## Format

A data frame with 5000 rows and 6 variables:

- ident:

  identifier, 1 or 2

- Y1:

  outcome 1 with 2 levels, observed for ident=1 and unobserved for
  ident=2

- Y2:

  outcome 2 with 4 levels, observed for ident=2 and unobserved for
  ident=1

- X1:

  covariate 1, integer

- X2:

  covariate 2, integer

- X3:

  covariate 3, integer

randomly generated
